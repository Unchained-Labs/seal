## [1.0.0] - 2026-04-01

### Added
- Voice-first task composer with push-to-talk recording.
- Kanban workflow view for queued/running/done/failed jobs.
- Runtime controls and per-task command tooling to interact with workspace execution.
- Task dependency selection and visibility to model multi-step workflows.

### Changed
- Improved resiliency and UX safeguards for voice submission timeouts.
